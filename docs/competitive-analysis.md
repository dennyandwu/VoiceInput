## VoiceInput 核心优势
1. **SenseVoice 中文识别更准** — 阿里达摩院专为中文优化，比 Whisper 中文场景表现更好
2. **极速识别** — RTF < 0.01，5秒语音49ms识别，远快于 Whisper
3. **开源免费** — 无订阅、无付费墙
4. **轻量** — int8 模型仅228MB
